# 🗄️ SQL Analysis

This folder contains SQL queries used to analyze the Brazilian e-commerce dataset.

## 📊 Key Analysis Performed
- Total number of orders
- Total revenue generated
- Top 5 product categories by revenue
- Monthly revenue trends
- Delivery delay analysis

## 🛠️ Tools Used
- MySQL

## 📌 Purpose
These queries were used to extract business insights and support dashboard creation in Power BI.
